## 一、项目概述

### 1.1 产品定位
一个纯展示型个人博客，专注于视觉效果的展示，采用玻璃态3D风格设计。

### 1.2 核心价值
- **视觉冲击力**: 玻璃态设计 + 3D流体背景
- **技术展示**: 展示前端开发能力
- **个人品牌**: 建立专业形象

### 1.3 目标用户
- 技术人员、设计师
- 潜在雇主/客户
- 开源社区项目协作者

### 1.4 技术栈
- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **UI组件**: shadcn/ui
- **3D渲染**: React Three Fiber
- **动画**: Framer Motion
- **部署**: Vercel

## 二、信息架构 (Information Architecture)

### 2.1 站点地图

```
个人博客
├── 首页 (/)
│   ├── Hero区域（个人介绍 + CTA）
│   ├── 最新文章（3列网格）
│   └── 页脚
│
├── 博客详情页 (/blog/[slug])
│   ├── 文章标题与元信息
│   ├── 文章内容
│   ├── 侧边栏（目录 - Desktop）
│   └── 阅读进度条（顶部）
│
└── 关于页 (/about) [可选 - MVP外]
    ├── 个人信息
    ├── 技能标签
    └── 联系方式
```

### 2.2 导航结构

#### 主导航（桌面端）
- 首页 (Home) → `/`
- 博客 (Blog) → `/` (首页即展示博客)
- 关于 (About) → `/about` [可选]

#### 移动端
- 汉堡菜单触发侧边栏导航

## 三、用户故事 (User Stories)

### 3.1 访客浏览博客

**作为** 访客  
**我想要** 在首页看到最新文章列表  
**以便** 快速了解博客内容

**验收标准**:
- [x] 首页展示所有博客文章卡片
- [x] 卡片显示标题、摘要、标签、日期
- [x] 卡片使用玻璃态设计
- [x] 悬停时有上浮 + 发光效果
- [x] 点击卡片跳转到详情页

### 3.2 访客阅读文章

**作为** 访客  
**我想要** 点击文章卡片查看完整内容  
**以便** 阅读完整文章

**验收标准**:
- [ ] 文章详情页展示完整内容
- [ ] 显示阅读进度条
- [ ] Desktop端显示文章目录
- [ ] 支持Markdown渲染
- [ ] 代码块有语法高亮

### 3.3 视觉体验

**作为** 访客  
**我想要** 看到流畅的3D背景和动画  
**以便** 获得沉浸式体验

**验收标准**:
- [x] 全页面3D流体背景
- [x] 毛玻璃卡片效果
- [x] 页面滚动时有平滑动画
- [x] 页面切换有过渡效果
- [ ] 鼠标交互反馈（粒子跟随）

## 五、非功能需求 (Non-Functional Requirements)

### 5.1 性能
- **首页加载时间**: < 2秒 (LCP)
- **3D渲染**: 60fps
- **移动端优化**: 降低粒子数量
- **图片优化**: 渐变占位避免图片加载

### 5.2 视觉设计
- **色彩系统**: 深色主题 + 渐变高亮
- **玻璃态效果**: backdrop-blur + 半透明背景
- **发光效果**: box-shadow 辉光
- **响应式**: Mobile / Tablet / Desktop

### 5.3 可访问性
- **键盘导航**: Tab键支持
- **Focus状态**: 可见焦点边框
- **动画尊重**: 支持 reduced-motion

### 5.4 浏览器兼容性
- Chrome/Edge: ✅ 完整支持
- Firefox: ✅ 支持（部分3D效果）
- Safari: ✅ 支持（backdrop-filter兼容）
- Mobile: ✅ 支持（优化性能）

```typescript
interface Post {
  slug: string;              // URL路径
  title: string;             // 文章标题
```
